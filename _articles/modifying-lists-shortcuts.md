| Action               | Shortcut       |
|----------------------|----------------|
| Expand/collapse      | `Ctrl+.`       |
| Expand/collapse all  | `Ctrl+Shift+.` |
| Toggle checklist     | `Ctrl+Alt+C`   |
| Toggle numbered list | `Ctrl+Alt+N`   |
