| Action                         | Shortcut           |
|--------------------------------|--------------------|
| Open help                      | `Ctrl+?`           |
| Exit search/file finder/search | `Esc`              |
| Toggle file pane               | `Ctrl+Shift+F`     |
| Toggle bookmark pane           | `Ctrl+Shift+B`     |
| Toggle checked items           | `Alt+C`            |
| Toggle notes                   | `Alt+N`            |
| Toggle spellchecking           | `Ctrl+Alt+S`       |
