| Action             | Shortcut         |
|--------------------|------------------|
| Bold               | `Ctrl+B`         |
| Italic             | `Ctrl+I`         |
| Code               | `Ctrl+Backtick`  |
| LaTeX              | `Ctrl+L`         |
| Make heading 1/2/3 | `Ctrl+Alt+1/2/3` |
| Clear heading      | `Ctrl+Alt+0`     |
