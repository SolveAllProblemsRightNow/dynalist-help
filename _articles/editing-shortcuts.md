| Action                       | Shortcut               |
|------------------------------|------------------------|
| Indent                       | `Tab`                  |
| Unindent                     | `Shift+Tab`            |
| Delete item                  | `Ctrl+Shift+Backspace` |
| Switch between item and note | `Shift+Enter`          |
| Mark as done                 | `Ctrl+Enter`           |
| Start new line               | `Ctrl+Shift+Enter`     |
| Undo                         | `Ctrl+Z`               |
| Redo                         | `Ctrl+Y`               |
